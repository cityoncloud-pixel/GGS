# GGS Kit 开发（Codex）

在本仓库开发 GGS 时，用户说「运行 GGS」请 Read：

- `package/templates/project_control/.ggs/templates/runner.prompt.md`
- `package/templates/project_control/.ggs/templates/grill.prompt.md`

目标项目测试请先 `ggs init -TargetPath <path>`，再对**目标项目**执行 GGS，而非只改 kit 模板。

CLI：`ggs agent -TargetPath <path> -Runtime auto`
